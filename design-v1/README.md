| 파일 | 역할 |

## 사용자별 노출 범위

| 사용자 종류 | 노출되는 mock | 비고 |

| **B2C (일반 사용자)** | ① ~ ⑧ + ⑯⑰ | 전문가/관리자 페이지는 절대 안 보임 |

## 열어보는 법

각 HTML 파일을 더블클릭하면 브라우저에서 바로 렌더됩니다.
폰트·이미지는 외부 의존 없이 인라인으로 들어가 있어 오프라인에서도 100% 열립니다.
