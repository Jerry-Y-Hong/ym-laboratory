# 🩺 K-Medi 바이오피드백-리추얼 통합 경험 모델 설계서
## (Digital Bio-feedback & Traditional Ritual Hybrid Platform Model)

본 설계서는 디지털 헬스케어 기술(스마트폰 카메라 PPG 센싱 기반 바이오피드백)과 전통 한방의 오감 자극 리추얼(Ritual)을 결합하여, 고객이 즉각적인 신체 치유 효과를 데이터로 직접 확인하고 바이오 데이터에 기반한 고도의 맞춤형 케어를 경험하도록 돕는 플랫폼 아키텍처를 정의합니다.

---

## 1. 전역 경험 루프 (Core Experience Loop)

이 모델은 단순히 차나 음식을 파는 것을 넘어, **"진단 ➡️ 조율(음용 및 리추얼) ➡️ 검증"**의 닫힌 루프(Closed-loop)를 스마트폰 앱을 통해 실현합니다.

```
[Pre-Measure: 진단]
고객이 스마트폰 카메라에 손가락 접촉 (PPG 광혈류 측정)
   ⬇ (10초 측정)
[AI 스트레스 & 피로 지수 산출]
자율신경(교감/부교감) 균형도 및 스트레스 지수 분석
   ⬇ (MFCO 추론 연동)
[Optimal Prescription: 추천]
맞춤형 한방 에센스 샷 + 리추얼 툴 (예: 릴렉스 드롭 + 스팀 안대) 매칭
   ⬇ (가맹점 즉시 서빙)
[The Ritual: 치유 음용]
아로마 브리딩과 리추얼 툴을 활용한 15분간의 시그니처 힐링 세션
   ⬇ (음용 완료 및 타이머 종료)
[Post-Measure: 검증]
스마트폰 카메라 재접촉 (PPG 재측정) ➡️ 자율신경 안정 및 스트레스 지수 감소율 시각화
```

## 2. 모바일 바이오피드백 기술 사양 (Mobile Bio-feedback Spec)

### 1) 스마트폰 카메라 PPG (Photoplethysmography: 광혈류측정) 연동
* **작동 원리**: 스마트폰 카메라 렌즈 위에 손가락을 대고 플래시를 켜면, 혈류의 흐름에 따라 손가락 끝의 투과 광량이 변합니다. 이를 이미지 센서로 초당 30프레임 이상 캡처하여 심박변이도(HRV: Heart Rate Variability) 데이터를 추출합니다.
* **측정 지표**:
  * **SDNN (심박동 표준편차)**: 신체 스트레스 대항 능력 (수치가 낮을수록 스트레스 상태).
  * **LF/HF Ratio (교감/부교감 신경 비율)**: 자율신경 균형 상태 (LF가 높으면 불안/긴장, HF가 낮으면 무기력/피로).
  * **스트레스 지수 (Stress Index)**: 위 지표들을 조합하여 0~100의 점수로 환산.

### 2) 데이터 매핑 및 MFCO 추론 연동
* PPG 측정을 통해 도출된 자율신경 상태 지수는 MFCO API의 입력 상태 파라미터(`STATE_ID`)로 변환되어 전달됩니다.
  * *LF/HF Ratio 과다(교감 과흥분)* ➡️ `ST-018 (정신적 긴장 및 수면 불안)` 자동 매핑 ➡️ **K05 안심안정 에센스 및 국화 스팀안대 추천**
  * *SDNN 극저(만성 피로)* ➡️ `ST-001 (기력 저하 및 육체 피로)` 자동 매핑 ➡️ **K01 기력보강 활력에센스 및 어깨 온기 찜질팩 추천**

## 3. 리추얼 키트(Ritual Kit) 규격화 및 플랫폼 유통

플랫폼 본사가 가맹점에 유통하는 스마트 리추얼 패키지의 원스톱 물류 규격입니다.

| 리추얼 도구 | 물리적 제형/사양 | 에센스 매칭 시너지 | 매장 서빙 가이드 |
| :--- | :--- | :--- | :--- |
| **스팀 아이마스크 (Steam Eye Mask)** | 개별 밀봉 파우치. 뜯는 즉시 공기 중 산소와 반응하여 40℃ 미온열과 천연 국화 향 15분간 발생. | **K05 안심안정 에센스** / 안구 청화 에이드 | 차가운 음료 서빙 시 컵 옆에 트레이에 얹어 함께 제공. |
| **허브 온기 팩 (Warm Herb Pack)** | 매장용 온장고 보관. 대황, 건강, 쑥 등 천연 허브가 내장된 패브릭 어깨 숄 형태. | **K01 기력보강 활력에센스** / 디톡스 온기 에이드 | 따뜻한 음료 서빙 전 고객 어깨에 직접 올려 감싸줌. |
| **지압 압박 링 (Acupressure Ring)** | 신축성 있는 금속 스프링 링. 손가락에 끼워 가볍게 굴리는 지압 도구. | **K02 순환온기 분말키트** / 활력 에이드 | 음료 컵 손잡이 또는 홀더에 액세서리 형태로 끼워 제공. |

## 4. 확장형 스마트 진단 게이트웨이 및 표준 API 연동 (Scalable Diagnostics Gateway)

스마트폰 카메라 PPG 측정 외에도, 사용자가 현재 진행 중인 다양한 외부 스마트 웰니스 디바이스나 특수 한방 진단 장비들과 원활하게 데이터를 주고받을 수 있도록 **확장 가능한 표준 API 게이트웨이**를 제공합니다.

### 1) 글로벌 웨어러블 헬스 앱 연동
모바일 앱이 운영체제(OS) 수준의 헬스케어 저장소와 동기화하여 백그라운드에서 생체 데이터를 수집합니다.
* **iOS (Apple HealthKit) 연동**: 애플워치가 주간/야간에 주기적으로 측정한 HRV(심박변이도) 및 안정 시 심박수(RHR) 수신.
* **Android (Google Health Connect) 연동**: 갤럭시워치/갤럭시링이 측정한 수면 깊이(Sleep Quality), 스트레스 데이터 동기화.

### 2) 외부 웰니스/의료용 IoT 장비 연동 프로토콜 (JSON Schema)
가맹점 매장 내 배치될 맥진기(Pulse Diagnosis Device), 체성분 분석기(InBody), 한방 생체 임피던스 측정 장치 등이 측정 결과를 플랫폼 서버로 직접 전송할 수 있도록 오픈 API 엔드포인트를 제공합니다.

```json
{
  "api_version": "v1.0",
  "device_id": "PULSE-DIAG-091A",
  "client_id": "USER-9941-KIM",
  "timestamp": "2026-06-02T10:22:00Z",
  "vital_signs": {
    "heart_rate_variability": {
      "sdnn_ms": 42.5,
      "rmssd_ms": 31.2
    },
    "autonomic_nervous_system": {
      "sympathetic_index_lf": 62.4,
      "parasympathetic_index_hf": 37.6,
      "ratio_lf_hf": 1.66
    },
    "body_composition": {
      "body_fat_percentage": 22.4,
      "skeletal_muscle_mass_kg": 31.5
    }
  }
}
```
* 시스템은 전송받은 외부 생체 원시 데이터를 기반으로 상태 코드(`ST-001 ~ ST-018`)를 재연산하여 맞춤형 식단 및 리추얼 처방을 동일하게 수행합니다.

## 5. 특허 청구항 보완 전략 (BM Patent Claims)

이 하이브리드 바이오피드백 모델은 비즈니스 모델(BM) 특허의 가장 핵심적인 방어 장벽이 됩니다.

* **독점 특허 청구 범위**:
  1. 사용자 단말기(스마트폰)의 광학 이미지 센서와 플래시 광원을 제어하여 사용자의 모세혈관 광혈류(PPG) 신호를 캡처하거나, 연동된 외부 웨어러블 디바이스 및 생체 측정 IoT 장비로부터 자율신경 변이 데이터를 표준 게이트웨이 API를 통해 수신하는 단계.
  2. 수신된 생체 데이터로부터 자율신경 균형도(LF/HF Ratio) 및 스트레스 지수를 산출하고, 이를 한의학적 기미론 기반 온톨로지 지식 베이스(MFCO)의 신체 취약점 상태 코드로 자동 매핑하는 단계.
  3. 매핑된 상태 코드에 부합하도록 기 결정된 약선 키트 포뮬러 및 오감 자극형 리추얼 툴(스팀안대, 찜질팩 등)을 가맹점 매장 단말기로 즉각 지시하여 제공하게 하는 단계.
  4. 상기 약선 키트 및 리추얼 툴 사용 완료 신호를 수신한 후, 단말기의 이미지 센서를 재구동하거나 외부 연동 디바이스로부터 자율신경 데이터를 재수신하여 자율신경 균형도의 개선 폭을 정량적으로 산출하고 이를 시각화된 리포트로 피드백하는 단계를 포함하는 스마트 웰니스 플랫폼 제공 방법.
