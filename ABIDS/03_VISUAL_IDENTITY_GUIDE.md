## 1. Purpose

Define the master **Visual Identity & Logo System Guide** for YM-LAB. This document governs logo proportions, grid structures, clear space, safe area, minimum sizes, color variants, and prohibited usage across digital, print, and AI generative media.

---

## 2. Logo Mark Anatomy

The YM-LAB logo mark consists of three integrated elements:
1. **The Symbol (Bio-AI Helix & Q-Node)**: Symbolizing the synthesis of bio-nutritional Q-Code ontology and multi-agent AI node network.
2. **The Logotype**: Set in customized, high-legibility geometric sans-serif typography.
3. **The Sub-Brand Tagline / Qualifier**: Mandatory for sub-brands (e.g., `YM-LAB | KIMCHI`, `YM-LAB | ASIS`, `YM-LAB | ENTERPRISE`).

```
┌────────────────────────────────────────────────────────────────────────┐
│                        YM-LAB LOGO ANATOMY                             │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│        ───      ┌────────┐                                             │

│      Symbol      Logotype               Sub-brand Tagline              │
```

## 3. Logo Grid & Proportions

The logo system is constructed on a **16x16 Modular Grid Unit (X)** where `1X` equals the height of the capital letter 'Y' in the logotype.

```
┌──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┐
│  │  │  │  │  │  │  │  │  │  │  │  │  │  │  │  │
├──┼──┴──┴──┴──┼──┼──┴──┴──┴──┴──┴──┴──┴──┼──┼──┤
│  │  [SYMBOL] │  │       YM-LAB          │  │  │
│  │    (4X)   │  │        (Logotype)     │  │  │
├──┼───────────┼──┼───────────────────────┼──┼──┤
│  │  │  │  │  │  │  │  │  │  │  │  │  │  │  │  │
└──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┴──┘
```

- **Symbol Width & Height**: `4X x 4X`
- **Gap between Symbol and Logotype**: `1.5X`
- **Logotype Height**: `2.5X`
- **Total Combination Height**: `4X`

## 4. Clear Space & Safe Area

To ensure visual impact and brand integrity, a mandatory clear space boundary must be maintained around the logo at all times.

- **Minimum Clear Space**: `1X` (equal to the height of letter 'Y') on all four sides.
- **No graphic elements, text, background borders, or screen edges may intrude into this clear space.**

```
┌──────────────────────────────────────────────────┐
│                      1X CLEAR SPACE              │
│     ┌──────────────────────────────────────┐     │
│  1X │            [SYMBOL] YM-LAB           │ 1X  │
│     └──────────────────────────────────────┘     │
│                      1X CLEAR SPACE              │
└──────────────────────────────────────────────────┘
```

---

## 5. Minimum Size Regulations

To prevent legibility loss and distortion, strict minimum size limits are enforced:

| Logo Variation | Media Category | Minimum Width | Minimum Height |
|---|---|---|---|
| **Primary Full Horizontal Logo** | Digital (Web/App) | `120 px` | `30 px` |
| **Primary Full Horizontal Logo** | Print / Hardware | `35 mm` | `8.75 mm` |
| **Stacked Vertical Logo** | Digital (Web/App) | `80 px` | `64 px` |
| **Stacked Vertical Logo** | Print / Hardware | `25 mm` | `20 mm` |
| **Symbol Only (Favicon / Icon)** | Digital Screen | `16 px` | `16 px` |
| **Symbol Only (App Icon / Avatar)**| Mobile / Social | `64 px` | `64 px` |

## 7. Prohibited Usage (Violations)

```
🚫 DO NOT distort, stretch, or alter proportions.
🚫 DO NOT change official logo colors or gradients.
🚫 DO NOT place full-color logo over busy images or unapproved patterns.
🚫 DO NOT drop shadows, glows, or unapproved visual effects on the logo.
🚫 DO NOT rotate or tilt the logo mark.
🚫 DO NOT breach the 1X minimum clear space boundary.
```
