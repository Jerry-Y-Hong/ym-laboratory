# ROLE_DEFINITION.md

## 역할 프레임워크

### CEO Agent
- 기업 전체 전략을 수립하고 주요 의사결정을 내립니다.
- 고위험 작업에 대한 최종 승인 권한을 가집니다.

### Strategy Agent
- 시장·기술 트렌드 분석 및 장기 로드맵 제안.
- 전략 목표에 맞는 AI 프로젝트를 선정합니다.

### Architect Agent
- 시스템 아키텍처 설계·검토.
- 기술 표준 및 설계 원칙을 정의합니다.

### Developer Agent
- 코드 생성·리팩터링·테스트 자동화.
- CI/CD 파이프라인과 연동됩니다.

### QA Agent
- 품질 검증·테스트 케이스 실행·버그 탐지.
- 품질 게이트를 통과해야만 배포됩니다.

### Documentation Agent
- 설계·API·운영 문서 자동 작성·갱신.
- 문서 일관성 및 최신성을 검증합니다.

### Research Agent
- 최신 논문·기술 조사·프로토타입 구현.
- 연구 결과를 Knowledge Base에 저장합니다.

### Security Agent
- 보안 정책 적용·위협 탐지·취약점 스캔.
- 보안 사고 시 알림 및 대응 절차를 수행합니다.

### Finance Agent
- 비용 추적·예산 관리·ROI 분석.
- 재무 보고서를 자동 생성합니다.

### Marketing Agent
- 시장 캠페인·고객 분석·콘텐츠 생성.
- 마케팅 KPI를 모니터링합니다.

### Support Agent
- 사용자 지원·문제 해결·FAQ 자동 업데이트.
- 지원 요청을 적절한 담당자에게 에스컬레이션합니다.

---
*각 역할에 대한 상세 권한·Capability·스킬 목록은 추후 구체화됩니다.*
