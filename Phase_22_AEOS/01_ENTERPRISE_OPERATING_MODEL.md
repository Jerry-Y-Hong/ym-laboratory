# 01_ENTERPRISE_OPERATING_MODEL.md

## Overview
이 문서는 **YM‑LAB AI Autonomous Enterprise Operating System (AEOS)** 의 전사 운영 모델을 정의한다. 전략, 조직, 비즈니스 프로세스, 데이터 흐름을 고수준에서 서술하며, 기존 **ADF v2.0** 아키텍처와의 연계성을 명시한다.

## Core Components
- **Strategy Layer**: 비전·목표·핵심 KPI 정의
- **Organization Layer**: 역할·책임·AI 운영 에이전트 매핑
- **Process Layer**: 주요 비즈니스 프로세스와 자동화 흐름
- **Data Layer**: 데이터 수집·분석·피드백 루프
