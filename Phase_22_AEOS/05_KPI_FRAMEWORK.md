# 05_KPI_FRAMEWORK.md

## Overview
AEOS의 핵심 성과지표(KPI) 프레임워크를 정의한다. KPI는 **전략·운영·기술** 세 축으로 구분되며, 각 지표는 측정 방법, 데이터 소스, 보고 주기를 명시한다.

## KPI Categories
- **Strategic KPI**: 비전 달성률, 시장 점유율, ROI
- **Operational KPI**: 시스템 가용성, 평균 복구 시간(MTTR), 배포 성공률
- **Technical KPI**: AI 모델 정확도, 데이터 처리량, 비용 효율성

## Measurement & Reporting
| KPI | Metric | Data Source | Frequency |
|-----|--------|------------|-----------|
| 비전 달성률 | % | 전략 보고서 | Quarterly |
| 시스템 가용성 | % | Monitoring 플랫폼 | Daily |
| AI 모델 정확도 | % | 모델 평가 레포트 | Per Release |

## Governance
- KPI Owner는 각 부문 책임자가 지정한다.
- KPI 리뷰는 **08_CONTINUOUS_IMPROVEMENT.md**와 연계하여 매월 수행한다.
