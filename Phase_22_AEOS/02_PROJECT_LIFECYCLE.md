# 02_PROJECT_LIFECYCLE.md

## Overview
AEOS 프로젝트의 전반적인 라이프사이클을 정의한다. 단계는 **Initiation → Planning → Execution → Monitoring → Closure** 로 구성되며, 각 단계는 기존 **ADF v2.0** 의 Gate 정의와 일치한다.

## Phases & Gates
| Phase | Gate | Description |
|-------|------|-------------|
| Initiation | Gate 1 | 프로젝트 목표 정의 및 초기 요구사항 수집 |
| Planning   | Gate 2 | 상세 설계·리소스 계획 (Phase 22 전용) |
| Execution  | Gate 3 | 구현·배포 (Phase 22 Deliverable 수행) |
| Monitoring | Gate 4 | 운영 모니터링·지속 개선 |
| Closure    | Gate 5 | 최종 검증·동결 선언 |
