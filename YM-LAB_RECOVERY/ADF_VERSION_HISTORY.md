## 1. Overview & Purpose

This document maintains the official **Architecture Development Framework (ADF) Version History** for the YM-LAB Enterprise Ecosystem. It documents framework evolution, governance standard updates, phase transitions, and freeze certification milestones from Phase 00 through Phase 30.

In compliance with the **YM-LAB Version Governance Policy** (`01_VERSION_GOVERNANCE_POLICY.md`), this registry explicitly distinguishes the **ADF Governance Version** (the framework standard) from the **Repository Release Version** (the project milestone state).
