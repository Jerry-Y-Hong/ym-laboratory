# 06_KOREAN_FOOD_CLASSROOM

## 문서 목적

한국 음식과 식문화를 누구나 쉽고 재미있게 이해할 수 있도록 교육 콘텐츠의 표준 구조를 정의한다.

이 문서는 단순한 레시피 소개가 아니라 음식에 담긴 역사, 문화, 과학, 건강, 지역성, 계절성을 함께 전달하는 것을 목표로 한다.

---

# 교육 원칙

① 쉽고 재미있게

② 스토리 중심

③ 음식보다 사람 중심

④ 문화와 역사 연결

⑤ 과학적 근거 병행

⑥ 실생활 활용 가능

⑦ 한국인의 시각이 아닌 세계인의 시각으로 설명

# 08_AUTOMATION_RULE

## 문서 목적

콘텐츠 제작 과정에서 반복되는 작업을 표준화하고 자동화하기 위한 규칙을 정의한다.

---

# 자동화 대상

- 콘텐츠 초안 생성
- 제목 생성
- 요약 생성
- 태그 생성
- 이미지 프롬프트 생성
- SNS 문구 생성
- 번역
- 교정
- SEO 점검

---

# 자동화 원칙

- 사람의 검토를 반드시 거친다.
- 사실 검증을 우선한다.
- 동일한 품질을 유지한다.
- 반복 작업을 최소화한다.
- 수정 이력을 관리한다.
